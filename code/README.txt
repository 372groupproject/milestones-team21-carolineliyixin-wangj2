p3_boolean.lua, p3_nil.lua, p3_number.lua, p3_string.lua and p3_table.lua are use to represent the different data type in Lua. Use command lua p3_fratue.Lua can directly compile and run the program and see the output.

p3_function.lua is function data type and need use command lua -i p3_function.lua and then use print(sum_even(n))(n is the random number you choose). The result is the sum of the even number small then the number you choose.

----------------------------------milestone 4 --------------------------------------
p4_module.lua, p4_module2.lua, p4_module3.lua, p4_environment.lua are the code examples that illustrate how subprogram works in Lua.
p4_module.lua talks about a basic approach to define packages and way to use it in another file.
p4_module2.lua talks about the way to write local functions in a package and only makes the necessary ones public (store into the global table).
p4_module3.lua is an example program that write a single "class" BankAccount and defines an attribute and two functions.
p4_module4.lua is a simple example showing how to write package in a file and use in another file.
p4_environment.lua talks briefly about a special feature in Lua called the global environment where  the programmer can access all global variables (all functions and libraries).
